# 1. 本日のミッション

第1回では、画像検査システムの目的、検査対象、良品・不良品の条件について調査しました。

画像検査では、AIカメラの性能だけでなく、次のような物理条件が判定精度に大きく影響します。

```text
背景の色
照明の当たり方
カメラとワークの距離
カメラの角度
ワーク停止位置
カメラ固定の強さ
```

つまり、今回の本当のテーマは、

```text
AIに頑張らせるのではなく、
AIが判定しやすい現場環境を設計する
```

ことです。

## ステップ0：安全確認

本日は、カッター、ハサミ、定規、段ボール、画用紙、テープ類を使用します。

以下を必ず守ること。

- カッターは必ずカッターマットまたは段ボール上で使う
- 刃を人に向けない
- 切る方向に手を置かない
- 大型カッターは教員の許可を得て使用する
- コンベアや可動部の近くで作業するときは、必ず電源状態を確認する
- 既設設備に穴あけ、接着、強い締め付けをしない
- 可動部にケーブルやジグが干渉しないようにする

## ステップ1：第1回仕様書フォロー

### 目的

第1回で書ききれなかった検査仕様書を完成させます。

### 作業内容

Teams上の報告書ファイルを開き、以下を完成させます。

| 項目 | 書く内容 |
|---|---|
| 導入 | なぜ画像検査システムが必要なのか |
| 背景 | 人間の目視検査だけでは何が難しいのか |
| 目的 | AIカメラで何を自動化したいのか |
| 良品の定義 | どの状態ならOKか |
| 不良品の定義 | どの状態ならNGか |
| 判定基準 | ズレ、欠品、赤色混入をどう見分けるか |

### 書き方の例

```text
良品：
青色部品、黄色部品が所定の位置に取り付けられており、
赤色部品が混入していない状態。

NG_ズレ：
部品が正しい位置から明らかにずれており、
HUSKYLENS画面上で部品の位置関係が良品と異なる状態。

NG_欠品：
本来あるべき部品が取り付けられていない状態。

NG_赤色混入：
本来使用しない赤色部品がワーク内に含まれている状態。
```

### 注意

「なんとなく違う」ではなく、  
**画像で見たときに何が違うのか** を書くこと。

## ステップ2：HUSKYLENS仮学習または画面確認

### 目的

背景や照明の違いが判定に影響することを確認するため、最低限の判定状態を作ります。

### 作業内容

すでに学習済みの場合は、その学習データを使います。  
未学習の場合は、簡易的に次の2分類だけを学習します。

| ID | クラス名 | 内容 |
|---|---|---|
| ID1 | OK | 良品 |
| ID2 | NG_RED | 赤色混入 |

この時点では、完璧な学習を目指しません。  
本日の目的は、**AIが背景や光に影響されることを体験すること** です。

### 注意

- 指や手を写した状態で学習させない
- 背景が大きく変わる状態で学習させない
- ワークは画面中央に置く
- レンズのプラスチック部分は無理に回さない

---

## ステップ3：環境限界テスト

### 目的

AIカメラがどのような環境変化で不安定になるかを確認します。

### 作業内容

ついたてを設置する前の状態で、以下の変化を与えます。

| テスト | 内容 | 観察すること |
|---|---|---|
| 背景変化 | ワークの奥で手を動かす | 判定が変わるか |
| 人の通過 | 背景側を人が通る | 誤判定が起きるか |
| 強い光 | スマホライトを当てる | 反射や白飛びが出るか |
| 暗くする | 手や紙で光を遮る | 判定が不安定になるか |
| 距離変化 | カメラを少し前後させる | ピントや認識が変わるか |

### 記録すること

| 条件 | 結果 | 気づき |
|---|---|---|
| 背景で手を動かした | 例：NGと判定された | 背景の動きをAIが拾った可能性 |
| スマホライトを当てた | 例：金属が光って見えた | 反射が強いと判定しにくい |
| カメラ距離を変えた | 例：ピントが甘くなった | 4〜5cmを保つ必要がある |

---

## ステップ4：背景ついたての試作

### 目的

背景の動きや余計な物体を隠し、AIがワークだけを見やすい状態にします。

### 支給資材

- 大型段ボール
- 色画用紙
- A3白紙またはスーパーファイン紙
- ハサミ
- カッター
- ガムテープ
- 定規

### 作業内容

段ボールをL字型に切り出し、自立するついたてを作ります。  
その表面に画用紙を貼り、背景色を変更できるようにします。

### 背景色の比較

以下の色を比較します。

| 背景色 | 観察ポイント |
|---|---|
| 白 | 明るいが、金属反射で白飛びしないか |
| 黒 | 輪郭が見えるが、影が強すぎないか |
| 水色 | 赤色部品が見分けやすいか |
| 緑 | 赤色との対比が強く出るか |
| グレー | 反射や色かぶりが少なく安定するか |

### 判断基準

背景色は、好みで選ぶのではなく、以下を基準に選びます。

```text
1. 赤色混入が見えやすい
2. 金属反射が強く出すぎない
3. HUSKYLENS画面でワークの輪郭が見える
4. 背景の動きが隠れる
5. OKとNGの見た目の差がはっきりする
```

## ステップ7：500円以内ジグ設計

### 目的

追加購入予算500円以内で、次回製作する固定ジグの案を作ります。

### 設計条件

必須条件は以下の通りです。

```text
1. HUSKYLENSのレンズ面からワークまで4〜5cmを保つ
2. カメラ角度が簡単にズレない
3. 背景ついたてがカメラ視野全体を覆う
4. コンベアやシリンダの動作に干渉しない
5. 既設設備に穴あけ・接着をしない
6. ケーブルが引っ張られない
7. 500円以内で追加購入できる
8. 取り外しても、できるだけ同じ位置に戻せる
```

### 使用可能なもの

- タミヤ ユニバーサルプレート
- タミヤ アーム・ステー
- スマホ用ホルダー
- Ulanzi MT-08三脚
- 段ボール
- 画用紙
- テープ類
- 100均で500円以内に収まる部材

### 100均購入候補の例

| 部材 | 期待する役割 |
|---|---|
| L字ブックエンド | 直角の土台、背景板の支え |
| 強力マグネット | 金属部への仮固定 |
| 小型クランプ | レーンや板の挟み込み |
| 配線クリップ | ケーブルの逃がし |
| 厚手カードケース | 背景板、遮光板 |
| すきまテープ | 振動吸収、位置決め |
| 結束バンド | ケーブル固定、部品固定 |

### 注意

500円は「安ければよい」という意味ではありません。  
限られた予算で、最も効果が高い部品を選ぶことが目的です。

購入リストには、必ず理由を書いてください。

例：

```text
購入希望：L字ブックエンド 1個 110円
理由：
背景ついたてを垂直に立てるため。
金属製で重さがあり、段ボールより倒れにくい。
タミヤプレートをテープまたは結束バンドで固定できる。
```

## ステップ8：ジグ設計スケッチ

### 目的

次回製作する「固定＋背景ついたて一体型ジグ」の設計図を作ります。

### 描くもの

ノートまたは白紙に、以下を描きます。

| 図 | 内容 |

- カメラのおよその高さ
- 背景板の幅
- 背景板の高さ
- 固定に使うネジ穴または固定位置
- 可動部からの距離
- ケーブルの逃がし方向

### 設計スケッチの考え方

きれいな図である必要はありません。  
ただし、次回その図を見て作業できる程度に、寸法と考え方が分かるようにしてください。

# 11. まとめ

本日の作業は、単なる工作ではありません。

画像検査システムでは、AIカメラの判定精度は、カメラだけで決まりません。  
背景、光、距離、角度、固定方法を設計することで、AIが安定して判断できる環境を作ります。

```text
AIを使う技術
ではなく、
AIを現場で使える状態にする技術
```

これが本日の中心です。
